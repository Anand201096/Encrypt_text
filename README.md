# Encrypt_text

- Very basic example of encryption using python.

## Requirements
- python3 ( 3.5 or above version )

## Installation
```
1) sudo git clone https://github.com/Anand201096/Encrypt_text.git

2) cd Encrypt_text

3) pip3 install -r requirements.txt

```

## Run
```
python3 encrypt_text.py 

```
